# CRUD para la consulta de placas con impresión de reporte

Desarrollado con :
* **Java 21**
* **Jasper Reports**
---

## Requisitos Previos
* Java SE Development Kit (JDK) 21.
* Configurar las variables de entorno en su IDE de desarrollo, puede ver las variables en el archivo application.properties.
* Para configurar la base de datos basica para este proyecto, puede revisar el archivo Placa.sql en la carpeta resources/datasource.
